# laravel-react-crud-api--ok
Testing Full Stack App (Laravel REST API + React.js SPA)
For React.js SPA:
npm install
Start React Project
npm run start

For Laravel API:
composer instal

Copy .env file
cp .env.example .env

Set Database Detail
DB_CONNECTION=mysql
DB_HOST=127.0.0.1
DB_PORT=3306
DB_DATABASE=
DB_USERNAME=
DB_PASSWORD=

Start Laravel Project
php artisan serve
